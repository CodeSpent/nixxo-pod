Installation settings

Hostname

If you are setting this up for a prod bot, choose escapepod.local. If you are setting up for OSKR/dev-only, IP address is recommended.


Weather API

Would you like to setup weather commands? This involves creating a free account at weatherapi.com and putting in your API key here. This is optional.


Knowledge Graph

Would you like to setup Knowledge Graph ("I have a question")? This involves creating a free account at houndify.com and entering your Client Key and Client ID.
Note: It may look like you only get a trial, but there is an actual free tier with 100 free requests per day.
This is optional.
